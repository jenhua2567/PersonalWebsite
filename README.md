# Jenny Hua — Personal Website

Personal website for Jenny Hua, Go-to-Market Strategist. Currently a coming-soon landing page.

## Stack

- Plain HTML/CSS — no build step required
- [Open Sans](https://fonts.google.com/specimen/Open+Sans) via Google Fonts
- Deployed on [Vercel](https://vercel.com)

## Local Development

```bash
python3 -m http.server 8080
```

Then open [http://localhost:8080](http://localhost:8080).

## Deployment

Push to the `main` branch. Vercel auto-deploys on every push.
